# Exercise 1: Prompt Builder

## Overview

In this exercise you'll learn to **"ground"** Einstein in your youth engagement records
so the assistant speaks only from facts, not hallucinations.

You will build two prompts:

- **1.1 Smart Youth Profile:** a Field Generation prompt that turns fragmented notes
  (plus the web) into a structured **Smart Youth Profile**.
- **1.2 Youth Outreach Email:** a Service Email prompt that turns that profile into a
  warm, personalized outreach email.

::: tip Prerequisites
Complete **Exercise 0: Pre-Setup Checklist** first — you need the Yi Zhuang Chew Contact
and engagement Case, Einstein enabled, and the Einstein permissions assigned.
:::

::: info Web & engagement enrichment
The profile is enriched from the **engagement notes in the record** and from the **web**
via a web search action (configured together during the session). Deeper engagement-data
enrichment can be layered on later via Data Cloud or an external service.
:::

Continue to **1.1 Smart Youth Profile** below.
