# Exercise 1: Prompt Builder

## Overview

In this exercise you'll learn to **"ground"** Einstein in your Salesforce Case records
so the assistant speaks only from facts, not hallucinations.

You will build two prompts:

- **1A — Email Prompt:** a Service Email prompt that ingests raw intake notes and
  outputs a professional, structured "Next Steps" email for the citizen.
- **1B — Field Generation Prompt:** a prompt that scans messy notes and automatically
  fills a structured **Generated Case Profile** field — saving 10–15 minutes of manual
  data entry per case.

::: tip Prerequisites
Complete **Exercise 0: Pre-Setup Checklist** first — you need the Carole Tan Contact
and Case, Einstein enabled, and the Prompt Template permissions assigned.
:::

Continue to **1A: Create an Email Prompt** below.
